# Learn a Bit

This repository contains a React/Vite learning app that can be packaged for mobile validation with Capacitor.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build

1. Build the web app:
   `npm run build`
2. Preview the production build:
   `npm run preview`
